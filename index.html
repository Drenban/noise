常见问题 (FAQ)
  • 为什么我们开发了 PeekX？

    我们希望为散户交易者打造一个简单而强大的量化工具。在金融市场中，机构凭借复杂技术占据优势，而散户往往难以企及。PeekX 是一个非盈利项目，旨在通过直观的数据分析和低门槛的量化功能，帮助普通交易者洞察市场，平衡这一差距。

  • 如何导入外部数据到 PeekX？

    最简单的方法是将数据（如 CSV 文件）复制粘贴到 PeekX 的数据导入界面。请注意，PeekX 目前仅支持标准格式的股价和交易数据，不支持非结构化文本或多媒体文件。详情请参阅 。

  • 如何分享我的量化策略？

    PeekX 不存储用户策略。如需分享，你可以在 PeekX 中创建策略后，将其导出为配置文件，然后通过社交平台或社区分享。确保配置文件包含必要参数，接收者可直接 。

  • 为什么我的策略没有显示在示例库中？

    PeekX 的示例库每天更新 4 次，从公开社区收集策略。请确保你的策略满足条件：
    在 PeekX 社区标记为 "公开示例"； 设置为 "公开 + API 可访问"。
    如果仍未显示，请检查网络连接或联系支持团队。

  • 为什么 PeekX 不支持复杂回测或多资产组合？

    为保持低成本和易用性，PeekX 将功能聚焦于个人量化基础需求，依赖本地计算而非服务器存储。复杂的回测或多资产组合需要更多资源，这会增加用户成本。我们建议高级用户结合其他工具使用 PeekX 的输出结果。

  • PeekX 支持移动端使用吗？

    是的，PeekX 通过响应式设计支持移动端浏览器访问。在手机或平板上打开 PeekX 网站即可体验完整功能，包括图表分析和策略编辑，无需额外安装应用。

关于

PeekX 项目源于对股票交易市场噪音过滤的假设，中文名 "窥见X" 寓意以更直观的视角洞察市场变化。我们旨在通过这一方式，提醒交易者关注市场噪音的存在及其影响。

在瞬息万变的交易环境中，价格波动常常受到短期投机、情绪驱动和信息过载的干扰，导致真实趋势被掩盖。PeekX 致力于剥离这些干扰因素，通过数据分析和可视化工具，为交易者提供清晰的市场信号。

我们的核心理念是将复杂的市场动态转化为简洁、直观的数据。PeekX 不仅是一个分析工具，更是一种思维方式，鼓励用户跳出噪音的迷雾，聚焦于真正驱动市场的力量。无论你是短线交易者还是长期投资者，PeekX 都能帮助你更冷静地解读市场，做出更理性的决策。

未来，我们计划进一步优化算法，整合多维度数据源，让"窥见"不仅限于股票市场，还能扩展至其他金融领域。通过持续创新，PeekX 希望成为交易者信赖的伙伴，助力他们在噪音中发现机会，在变化中把握确定性。
关于我们
PeekX 专注于为个人用户提供量化实现，而非面向机构用户。在当今金融市场中，机构凭借先进的量化技术和资源优势，大幅削弱了散户捕捉市场机会的能力。PeekX 旨在扭转这一趋势，通过简洁高效的工具，让更多散户以低成本参与个人量化交易。我们坚信，量化不应是机构专属的特权，而应成为每位交易者触手可及的利器。
